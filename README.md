## Introduction

《剧本杀》是一款以个人为中心的辅助游戏推理的应用。每名玩家都有自己的剧本，玩家需将自己代入到剧本角色中，同时辅助玩家进行剧本杀，引导剧本杀流程，同时期间不断收集线索，与其他玩家进行交流，认真推理最终找到真相。
